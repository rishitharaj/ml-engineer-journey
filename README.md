### Day 2: Linear Regression from Scratch
#### Objective
Implemented Linear Regression using only NumPy to understand how machine learning models learn from data.

#### Key Concepts Covered
- Forward pass (y = Wx + b)
- Mean Squared Error (Loss Function)
- Gradient computation
- Gradient Descent
- Training loop

#### What I Built
- Created synthetic dataset
- Implemented prediction function
- Computed loss using MSE
- Derived gradients manually
- Trained model using gradient descent

#### Key Learnings
- Model learns by minimizing error using gradients
- Importance of learning rate in convergence
- Training is an iterative optimization process

#### Results
- Model successfully learned the underlying pattern
- Loss decreased over epochs
- Visualized regression line fitting the data
